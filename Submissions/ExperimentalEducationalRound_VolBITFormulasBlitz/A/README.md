# A. Again Twenty Five!

  [問題はこちら](https://codeforces.com/problemset/problem/630/A)
  
- 概要<br>
  5 の n 乗の下 2 桁を出力せよ。<br>
  n は 2以上である。
  
- 発想<br>
  n が 2 以上の場合、5 の n 乗の下 2 桁は常に 25 になる。<br>
  
  
- コード（C++）

  ```cpp
  #include <bits/stdc++.h>
  using namespace std;

  int main() {

    long long n;
    cin >> n;

    cout << 25 << endl;

  }
  ```
  
- コード（Java）

  ```java
  public class Main {

    public static void main (String[] args) {

      System.out.println(25);

    }

  }
  ```
